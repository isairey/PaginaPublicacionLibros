# 🌌 Acerca de BookVerse

**BookVerse** es una plataforma de lectura y publicación digital diseñada para conectar lectores y escritores en un mismo espacio.

Los usuarios pueden publicar novelas, cuentos, poemas y libros completos, mientras que los lectores pueden descubrir contenido, seguir autores y crear bibliotecas personales.

La plataforma permite:

- 📚 Publicar libros digitales
- ✍️ Crear capítulos y series
- 🔍 Buscar libros por categoría
- ❤️ Guardar favoritos
- ⭐ Calificar obras
- 👤 Seguir autores
- 📖 Leer desde cualquier dispositivo
- 🌙 Modo oscuro para lectura

El proyecto fue desarrollado para practicar:

- React
- TypeScript
- Arquitectura Frontend Moderna
- Gestión de Estados
- UI/UX
- Diseño Responsive
- Plataformas de contenido digital

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/bookverse
```

---

## 2️⃣ Instalar dependencias

```bash
npm install
```

---

## 3️⃣ Configurar variables de entorno

```env
VITE_API_URL=
VITE_FIREBASE_API_KEY=
VITE_AUTH_DOMAIN=
```

---

## 4️⃣ Ejecutar proyecto

```bash
npm run dev
```
